An Integrated Machine Learning–Rock Physics–Seismic Inversion Workflow

for Reservoir Characterization: A Case Study of the Sawan Gas Field,

Pakistan



OVERVIEW:  The workflow uses well-log data to predict shear-wave travel

time (DT4S) from conventional logs using machine learning models. The

study is based on wells from the Sawan Gas Field, Pakistan.



REPOSITORY CONTENTS 

SAWAN\_SHEAR\_WAVE\_PREDICTION.ipynb : Main workflow

notebook 

test\_quick\_run.py : Minimal quick test for checking the code

setup 

requirements.txt : Python package requirements 

sample\_data/ :

Sample input data for testing 

outputs/ : Example output files

generated by the workflow



REQUIREMENTS Python 3.10 or later



Main Python packages: numpy pandas matplotlib seaborn scikit-learn

xgboost lightgbm catboost tabpfn lasio



INSTALLATION Install dependencies with:



pip install -r requirements.txt



QUICK TEST Run the quick test with:



python test\_quick\_run.py



This verifies that the main libraries are installed correctly and that a

simple machine learning regression example can run successfully.



MAIN WORKFLOW Open the notebook:



SAWAN\_SHEAR\_WAVE\_PREDICTION.ipynb



Then run the cells sequentially.



The notebook includes: 1. loading LAS well-log files 2. cleaning and

standardizing logs 3. exploratory data analysis 4. missing-value and

outlier handling 5. model training and comparison 6. prediction of DT4S

7\. export of cleaned and predicted results



INPUT DATA The workflow uses LAS well-log files from the Sawan Gas

Field. If the original dataset cannot be publicly shared, provide sample

data in the ‘sample\_data’ folder.



OUTPUTS Example outputs may include: - cleaned CSV files - predicted

DT4S logs - comparison plots - model performance summaries



Store representative output files in the ‘outputs’ folder.



COMPUTER CODE AVAILABILITY The code used in this study is publicly

available in this repository. It includes the workflow for data

preprocessing, machine learning model training, evaluation, and

prediction of shear-wave travel time from well-log data.
